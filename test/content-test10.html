Event Details

HTML5 and other New Exciting Web Technologies (as Bruce Lawson has termed CSS3, SVG, WebGL and other "not really HTML5 but still exciting web technologies") are whitehot right now.

So Web Directions has put together the NEWTup concept,  a meetup where you can learn the latest about HTML5, CSS3, SVG, geolocation, responsive design, and much more, connect with your peers in web design and development, and keep up to date with what's going on in the web Industry locally and around the world.

As with all Web Directions events, you know it will be a fun night with some great presentations, and there'll be a drink or two and a bite to eat.

The July NEWTup features two awesome presentations that will inspire you to get out there and "start buidling stuff!":

Real Time Programming with Arduino and JavaScript: Justin Maclean

The Arduino platform is a low cost, easy to program open source hardware and software platform than enables you to communicate in real time with the physical world. In this talk Justin Maclean will go through the steps involved to interact with Arduino hardware directly from a web browser using Javascript and web sockets.

Justin Maclean has managed his own consulting company Class Software for 15 years and over that time has worked on hundreds of web applications, mobile applications and desktop applications. He has been working with Javascript since IE3 days and with the Arduino platform for several years. He's spoken at more than a dozen conferences and run many training courses on Arduino and real time programming.  He is also an Apache Flex board member and committer, an Adobe Community Professional and a certified trainer for ColdFusion and Flex.

 WebGL - Alex Danilo

Alex Danilo from Google demos and walks-through some of the truly eye-popping things they have been doing with WebGL, including a bunch of stuff you definitely won't have seen anywhere before this evening. See it first at the July NEWTup!

Alex has spent over a decade on various W3C working groups developing the standards we all know and love, all while running a couple of start ups doing web engines for mobile and embedded markets. He's shown off his web technology based products at CES and NAB in Vegas, IBC in Amsterdam, multiple trade shows in Japan and now works at Google, spreading the good word about HTML5 and Chrome.
